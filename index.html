Rent-read-return

Titanium app developed for justbooksclc.com members to rent the books from smart phones

Download this project as a .zip file Download this project as a tar.gz file

This application has the authentication & authorization, searching and renting books subsystems. This application allows a member of justbooksclc to search and rent books with a simple gesture. This application can be run on any mobile/tablet platforms that support Appcelerator Titanium.
